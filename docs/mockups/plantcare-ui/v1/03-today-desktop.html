What's different on desktop, and what isn't

Same

Same palette, same core hero card, same personality voice system, same care animation, same progress ring, same data. Desktop is not a different app — it's the same app with more room.

Different — layout

Bottom dock → left sidebar. Single column → 2-column content below the hero (tasks main, widgets right). Hero plant scales from 230px → 340px with more presence.

Different — typography

Page titles get Fraunces italic serif for editorial weight ("Good morning, Robert"). Hero speech quote uses Fraunces too — 36px italic. Body text stays Plus Jakarta Sans.

Extra on desktop

Widget column gains room for "Since you were gone", "Your jungle" stats, and "Discover picks". These exist on mobile as separate screens or scroll sections; desktop shows them all at once.

Phase 3 impact

The 3D isometric Greenhouse view lives in the House screen on desktop — takes the full content area. Three.js with orbit controls. Mobile gets a simpler tap-to-drill version or stays on the grid.

Editorial flourishes

One hand-drawn dashed arrow curves above the hero (you can see it faintly behind Monty). Used sparingly — one or two flourishes per page max, never competing with content.
