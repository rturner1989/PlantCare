How each step adapts per intent

One wizard, four paths. Copy + emphasis + final CTA differ by the answer given in Step 1. Same steps in the same order; per-intent config drives content. User.onboarding_intent persists for future in-app defaults.

What the onboarding locks in

Eight steps for the happy path. Two for worried users. Minimal friction. Captures intent first so the rest adapts. Plays the game loop + journal from day one.

  • Intent captured in step 1. Four paths shape the subsequent copy, skips, and final CTA. Sick Plant forks the wizard entirely into a 2-step Doctor branch.
  • Multi-plant by default. Step 3 expects users with existing plants (Forgetful case) to add several. Add-another is primary. Search is backed by pg_search + Perenual fallback; popular grid seeds the most common picks.
  • Spaces cover indoor + outdoor. Two preset groups, same chip component. Custom spaces allowed.
  • Environment per-space. Plants inherit. Segmented (Low/Medium/Bright) uses the canonical paper-deep + paper-active + forest text treatment. Phase 2 swaps outdoor cards for season/frost controls.
  • Stakes step seeds the game loop visually. Two rings — streak at Day 0 (0%), vitality at 50% — render before any data exists. Trusted-but-empty state mirrors the Today header that'll load moments later. Promoted for Forgetful + Just Starting; skipped for Sick + Browsing.
  • Journal live from step 6. System events already created for each plant; user writes first entry if they want. Feature exists from day one, not promised for later.
  • Welcome-home CTA routes by intent. Forgetful → Today. Just starting → Today w/ "take your time" copy. Browsing → Library. One wizard, multiple exits.
  • Sick Plant bypasses the greenhouse setup entirely. Step A (symptom capture + photo + optional note) → Step B (review + diagnose) → Plant Doctor flow (mockup 33). Spaces / Plants / Environment / Stakes / Journal deferred; user fills those later at their own pace.
  • Motion vocabulary matches Today. Bottom-sheet dialogs for sub-flows, full-width right-slide drawers when organiser opens later. Circle motif throughout (logo, chips, avatars, rings, portrait frame).
  • Type scale aligned. Same 10-value scale as mockups 17 + 18. Fraunces italic for display headlines + scientific names + contextual hints; Plus Jakarta for everything else. No bespoke sizes.
