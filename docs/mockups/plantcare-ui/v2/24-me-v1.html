Me · default. Desktop lays out as: profile hero (avatar + name + email + Edit/Sign-out actions), 4-wide stat strip (Streak · Greenhouse · Rescues · Vitality — game-loop primitives), 2-col card grid (Intent · Notifications · Today layout · Preferences), danger zone at the bottom. Intent is prominent because changing it cascades — re-scores Today, retones the tour, re-weights notifications, picks different jobs-rail ordering. Notifications card uses row-style toggles (toggle-switch primitive) with a quiet-hours segment. Today-layout card is a drag-handle list with per-widget toggle — same primitives as the Notifications card so the user doesn't learn two UIs. Preferences covers display, units, plant-voice intensity (new — throttles quote frequency), week-start. Danger zone has its own card with a coral-dashed border + destructive buttons (export, password change, delete account). On mobile the 2-col grid becomes a 1-col stack, stats collapse to 2×2, sign-out moves into the danger zone stack as a secondary action, intent chips become 2×2. Everything else carries over.

Me locked

Me is the app's steering wheel. Intent at the top of the page because it's the highest-leverage control the user has — everything else (notifications, Today layout, schedule cadence) has a default sourced from it. Stat strip sells the game-loop positioning: streak + rescues + vitality are what a companion app measures, and putting them above the fold reinforces the "Nintendo-pets" frame of v2.

  • Profile hero. 120px avatar (gradient-brand + Fraunces italic initial) with a ✎ edit-badge nested on the bottom-right. Badge opens the full Edit profile bottom sheet/dialog (avatar · name · email · password) — one entry point, not a separate "Edit profile" button. Sign out sits as the single secondary action on desktop; on mobile it drops into the danger stack.
  • Stat strip = game-loop primitives. Streak (sunshine), Greenhouse (mint), Rescues (coral), Vitality (forest). 4-up on desktop, 2×2 on mobile. Each card: circular icon + uppercase label + Fraunces italic value. Rescue + streak feed the engagement loop; vitality is the aggregate health score.
  • Intent card — prominent. 4 chips (Just starting · Forgetful · Sick plant · Browsing) with personality emoji. Selected = mint fill + emerald border. Changing triggers a toast: "Re-scoring your Today…". Drives: landing destination, tour length, push aggressiveness, species filter, jobs-rail order, tip frequency.
  • Notifications card. Toggle-switch primitive + segment primitive (for Quiet hours). Rows: Care reminders · Rescue alerts · Milestones · Weather · Quiet hours. Each row has icon + title + meta. Reuses the toggle + segment vocabulary from the rest of the app.
  • Today layout — widget priority. Drag-handle list of widgets the user can pin to Today. Each row: ⋮⋮ drag handle + circular icon + name + meta + toggle. Intent sets a default order; user overrides stick.
  • Preferences card. Theme (Warm · Dark · Auto) · Units (Metric · Imperial) · Week starts (Mon · Sun, feeds Journal · Schedule + weekly calendar).
  • Danger zone. Coral-dashed outline card. Export data · Change password · Delete account (coral). On mobile, Sign out moves in alongside (desktop keeps it in the hero).
  • No voice library, no personality labels. Profile stays facts-only (name, email, stats). Personality lives in the plant-voice quotes the user's plants emit — not on the user's own profile.
  • Mobile dock. Me is the 5th primary slot (◉). Canonical template from mockup 18.
