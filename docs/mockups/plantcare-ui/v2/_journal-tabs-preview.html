Design notes. Tabs use rounded-top rectangles (14px) sitting atop the folder panel. Inactive tabs tilt ±0.6° for organic-stack feel; active tab flat. Tab count pill on right. Folder panel's top-left corner is sharp where the active tab attaches — asymmetric border-radius (0 14px 14px 14px). Paper-texture grain overlay on the panel sells the "document" metaphor. Content inside = journal entries with paper-deep entry cards; author + timestamp meta; mood words italicised via Fraunces + coral. Dashed divider under the folder head reads as a filing-card line.

On mobile: tabs become horizontal scroll carousel (same rotated aesthetic retained), folder panel full-width below.
