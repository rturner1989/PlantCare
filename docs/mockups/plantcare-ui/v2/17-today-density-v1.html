2026-04-24 · moderate polish pass — post-21 density baseline

Tightened vertical rhythm (main-col section gap 22 → 18px, section-label gap/margin 14 → 12px, stats gap 14 → 12px). Locked ritual-row slab height to 56px with centred flex so rows with vs without subtitles match. Reduced job-card padding (14×16 → 12×14) and added an .urgent variant — "Check on Monty" flags with coral-tinted icon + coral-deep title, matching House room-card urgency signalling. Radial-wheel primary spoke picks up a 2.2s wheel-pulse when Water is overdue, sharing vocabulary with mockup 22's hero wheel. Copy tone: "Your greenhouse" → What you've got, generic "· six of fifteen" → 4 of 15 things you can do with emerald-800 emphasis, UI narration on the collapsed-rail greeting swapped for the intimate companion phrasing. Tokens: dropped the stale --radius-xl from the palette, normalised the blob bleed blur to 70px (matches 21). Personality-chip 🎭 on the anniversary goal ring replaced with ✨ — anniversary is a milestone, not a personality type.

  • No more layout reshape. Previously the rail stole 300px of main column when expanded. Now it's position: absolute floating on top with backdrop-filter: blur(24px) saturate(1.4) + rgba(251,248,241,0.72) paper-tinted translucent bg. Dashboard stays full-width underneath.
  • Mac-style persistent trigger. Single small glass circle top-right of main when rail closed (mockup frame 3). Round, translucent, backdrop-blur, coral unread dot. Click opens the organiser. Always in the same spot — no more random floating pill.
  • Independent scroll. Dashboard column scrolls inside the main viewport area; organiser scrolls inside its own floating panel. Two overflow-y: auto containers, no interference. Scroll position preserved per-surface.
  • Calendar interactability — Phase 1 plan. Calendar is interactive:
    • Click a day → popover shows that day's rituals + option to "Add note / event" for that day.
    • Hover a day → small tooltip showing the ritual count.
    • Flag a date-range as "away" / "vacation" — suggests frontloading care before leaving, catches you up on return.
    • Phase 2: drag a ritual row from Today's list onto a future day to reschedule. Weather-aware conflict warnings.
    • Phase 3: recurring manual events (family birthdays, out-of-town, etc.) via the organiser.
  • Journal = organiser, codified. The Organiser panel (formerly "right rail") houses multiple cards — journal is one of them. Full Journal region expands from the rail toggle: Entries / Album / By plant / By mood / Milestones. Rail glance; region archive.
  • Glass + blur treatment. Kept recognisable (Mac Monterey vocabulary), not iOS-26 rim-highlight — that direction was retired 2026-04-22 and isn't coming back. The translucency + blur + inset white highlight + soft drop shadow = enough depth; no refraction, no edge-lighting gimmicks.
  • Animation on open/close. Framer Motion: initial={{ x: 40, opacity: 0 }} animate={{ x: 0, opacity: 1 }} exit={{ x: 40, opacity: 0 }} with { type: 'spring', stiffness: 260, damping: 28 }. Organiser trigger cross-fades to close button. Cards inside stagger-fade (40ms gap). Reduced-motion = pure opacity cross-fade.
  • Accessibility. role="dialog" aria-label="Organiser", focus trap while open, Esc to close, Tab cycles within. Click outside panel dismisses. Screen readers announce state changes.
