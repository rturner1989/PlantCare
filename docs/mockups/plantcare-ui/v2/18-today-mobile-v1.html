Three distinct motions on mobile — all shown. User learns "right = info, bottom = action, centre = wheel". Frame 2 · right-slide full-width drawer hosts the Organiser / notification centre — matches desktop's right-side placement; back arrow + ✕ dismiss. Frame 4 · Vaul-style bottom sheet hosts transient CRUD actions: Add / Edit / Delete / Log care / Add space. Drag-handle on top for swipe-down dismiss. Dedicated dialog library lives in mockup 20 covering all CRUD variants. Frame 3 · radial wheel at long-press anchor — for single-plant rich actions. Same visual language across all three (circles, warm paper, gradients), different gesture vocabulary per purpose.

Desktop vocabulary → mobile adaptation. Sidebar → bottom dock (5 slots, FAB raised centre). Right-rail glass overlay → right-slide drawer (same content, different container). Collection + jobs grids → horizontal scroll carousels with right-edge fade. Weather strip one-line. Weekly calendar 7-column grid. Rituals keep swipe-to-complete + handle-dots + 6px peek. Circles everywhere — rings 54px, portraits 96px, wheel 260px.

Mobile rules carried forward

What the iPhone frame establishes for the other six regions.

2026-04-24 polish pass. Ritual row padding rebalanced (10px top/bottom, flex-centered) so tag-less rows don't sag. Horizontal carousel primitive now feathers on both edges with a 16px linear-gradient mask + -16px negative margin, so items breathe at the card rail's left and right equally — post-21 canonical shape. Greet-sub warmed to match mockup 17's tone ("One needs you — the rest are thriving."). Confirmed clean against frozen rules: no --radius-xl, no hard-coded blur/radius px, no personality-badge chips, no voice-sample markup, Plus Jakarta + Fraunces only.

  • Sidebar → Dock. 5-slot bottom dock becomes the canonical mobile nav. Same 5 regions as desktop sidebar; FAB at centre slot, elevated.
  • Right-rail → bottom sheet. Whatever was in the glass rail on desktop becomes a bottom sheet on mobile. Tabs replace the card stack. Vaul-style drag-dismiss handle at top.
  • Horizontal multi-item rows become scroll carousels. No grid wrapping on mobile — keep content flowing horizontally.
  • All circles stay circular. Plant portraits, rings, FAB, dock icons, sheet avatars — all round. Motif survives the shrink.
  • Top bar + dock are both flush edge-to-edge with backdrop-filter translucency. No floating dock (that was v1 / abandoned), no floating top bar.
  • Swipe-to-complete + handle dots + permanent 6px peek carry over unchanged. iOS-native feel.
  • Radial wheel on tap-hold appears at tap point — not drawn in this mockup but same vocabulary as desktop. Will surface on the House + Plant Detail mobile mockups.
  • Dynamic Island + safe areas respected. 54pt top safe area, 34pt home indicator zone at bottom (30px dock bottom padding accounts for it).
