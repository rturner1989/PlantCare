Care tab (default). Hero: radial wheel with the medallion portrait at its centre (coral-ring-pulse when wilting/overdue — matches the mood signal on House room cards). Six spokes orbit the portrait: primary (Water · green normally, coral-pulse when urgent) + Feed · Photo · Note · Doctor · Move. Text column to the right leads with a plant-voice quote — Fraunces italic, coral quote-mark, attributed and timestamped. Below: Fraunces italic name, species italic, 2 meta pills (space · age). Four care-ring cards below: Water · Feed · Light · Mood. Three panel cards under that: Schedule · Recent care · Environment. On mobile the wheel scales to 300px (medallion 150px), hero stacks vertically, quote sits between species + meta pills, care rings become 2×2, panels stack.

Journal tab. Hero collapses to a compact 120px medallion + text block, freeing vertical space for entries. Nested file-tabs (Entries / Photos / Milestones) use the same vocabulary as the global Organiser — but filtered to this plant only. Entry types visually distinguish: system events (sunshine-tinted meta), user entries (emerald meta), milestone markers (also sunshine, with cake icon). Photo entries render thumbnail inline; tapping expands to full photo album view (Photos tab). Paper-texture panel sells the filing-folder metaphor consistently with the global Journal.

Plant Detail locked

  • Portrait medallion. 260px desktop / 180px mobile. Warm ivory background w/ grain, soft sunlight wash top-left, Perenual photo or species-glyph inside. Mood-ring pulses coral when wilting/overdue — matches House room-card urgent signal.
  • Plant-voice quote. First thing the user reads is a line spoken BY the plant, in Fraunces italic with a coral quote-mark. Personality + mood drive the line — dramatic Monstera wilting → melodramatic lament; stoic snake plant chill → terse acknowledgement; prickly cactus thriving → dry one-liner. Timestamped attribution ("— Monty, 9:41 this morning") reinforces the conceit. Rotation: Phase 1-2 pulls from a curated bank (personality × mood × care-state). Phase 2.5 (bot arrives) swaps the bank for Claude-generated lines — same slot, same voice, fresher content. If user lingers on the page >30s a new line can swap in with a fade.
  • Meta pills. Space · Age. Two at-a-glance facts. Personality is NOT labelled — inferred via the plant-voice quote + emotes + copy. Mood is NOT labelled — owned by the Mood ring.
  • Radial action wheel. Medallion sits at the centre of a six-spoke orbital wheel. Primary spoke (Water) is filled green and pulses coral when urgent. Satellites: Feed · Photo · Note · Doctor · Move. Same CSS vocabulary as mockup 17's long-press wheel — Plant Detail just shows it always-open as the primary per-plant action surface.
  • Four care rings. Water · Feed · Light · Mood. Concentric rings show progress/urgency at a glance. Water ring = coral when overdue. Label + value only; schedule detail lives in the Schedule panel.
  • Tabs. Care (default) · Species · Journal. Unified segment primitive (radius-md) — matches onboarding `.segmented` + House `.view-toggle`. Outer Photos tab dropped; Journal holds nested file-tabs (Entries / Photos / Milestones) which own all photo browsing for this plant.
  • Panel order: Schedule · Recent care · Environment. Clean role split — rings = current state, panels = config + history. No duplicated countdowns, no redundant labels.
  • Page-level scroll. Hero scrolls away as you page down; tab-bar sticks to the top so content earns most of the viewport. iOS-style large-title pattern.
  • Environment inheritance. Plants inherit from their Space's defaults. Override inline; changes persist to this plant only.
  • Long-press medallion = radial wheel. Same vocabulary as Today's plant cards. Quick-action menu without leaving the plant page.
  • Journal filtered per-plant. Everything in the global Organiser's journal, scoped to this one plant. Same vocabulary (system / user / milestone entry types, photo thumbnails inline), different filter.
  • Mobile stacks hero vertically. Medallion → name → meta pills → primary CTA. Care rings 2×2 below. Tab bar sticky. Panels stack.
  • Breadcrumb on desktop. House / Living Room / Monty · Journal. Provides path-back and current context. Mobile uses back-button on top bar instead.
