Treatment breakdown

Each technique's impact + which treatment it lives in. You can mix (e.g. "take B + radial glow from C but skip the tilt").

Technique What it does A B C
Grain / noise overlay Kills plastic feel of the solid gradient. Organic, film-like texture. Invisible unless you're looking; felt everywhere. no yes yes
Floating leaf silhouettes Botanical ambient depth. Large (~400px), low opacity, rotated at card corners and off-edge positions. Evokes overgrowth, abundance. no 2 3+
Gradient display headline Forest→emerald→leaf gradient on Fraunces italic. Richer than flat ink. Only on h2 display — never body copy. no yes yes
Dashed arrow flourish Editorial hand-drawn touch — lime dashed curve + arrowhead. Optional Fraunces italic annotation ("let's begin"). One per page max. no yes +label
Wavy line divider SVG stroke between headline and subhead. Soft, hand-drawn. Replaces hard padding-only separation. no no yes
Radial sunburst glow Lime radial gradient behind the illustration. Makes the plant feel spotlit. Subtle — 35% opacity at centre, fading to transparent by 70%. no no yes
Idle motion (leaves drift + pot sway) Background leaves rotate-translate on 18–22s loops. Pot sways 1.5° at 6s. All respect prefers-reduced-motion. Adds liveness without distraction. no no yes
Drifting sparkles 5 emoji/glyph sparkles at random positions, breathing 0→0.7→0 over 4s with stagger. Decorative. Also reduced-motion-safe. no no yes
Editorial asymmetry One word in the headline raised + rotated + recoloured. "your" in "Welcome to *your greenhouse*". Magazine-like. One per page maximum. no no yes
Tilted step eyebrow Numbered/named pill at the top of the card, rotated -4°. Pinned. Small ✦ prefix. Character touch — reads as "written on the card" not "UI label". no no yes
Layered card shadow Main card's shadow includes a second mint offset — looks like there's another card just behind it. Depth without content. no no yes
My recommendation. Somewhere between B and C, not all of C. Take B's grain + leaves + gradient headline + dashed flourish as the floor — applied to every screen. From C, cherry-pick radial glow, wavy dividers, and editorial asymmetry only on moment screens (Welcome, Step 7 Done, Today hero state, Plant Detail portrait). Leave idle motion, drifting sparkles, and layered-card shadow as Phase 1.5 polish — they make the app feel alive but add CSS complexity that'd slow initial implementation.

Also — RN parity. Most of these techniques port to React Native (gradients via react-native-linear-gradient, SVG via react-native-svg, grain via a texture image, motion via motion/react-native). Wavy dividers + gradient text on display are the fiddly ones for RN — plan to use font file weight swaps + image-mask workarounds when we get there.
