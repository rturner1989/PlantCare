What holds the system together

One motif drives every surface. The wheel isn't a gimmick — it's the thesis. Plants live in circles. Progress lives in rings. Actions bloom from rings. Even section markers are dots-with-orbits.

  • Circle as signature. Portrait medallions on Welcome + Plant Detail. Circular thumbnails on Today + photo album. Progress rings for streak + vitality + per-plant care rings. Section-anchor dots. Logo mark. FAB. Wheel. Spoke buttons.
  • Notification-centre pattern. Reused on Today's rituals, right-rail journal, Plant Detail journal tab. One "feed card" primitive, three homes.
  • Swipe row, fixed. Single surface, flush reveal, shared shadow.
  • Sidebar-only nav + floating signatures. No top bar. Bell + ⌘K hint + FAB carry global utilities distinctively.
  • Palette sharpened. Paper-ivory primary surfaces, ochre-tinted warm shadows, coral reserved for urgency / secondary accents, forest + leaf for brand + CTAs, sunshine for milestones.
  • Editorial type still does the heavy lifting. Fraunces italic for display + numeric moments; drop cap on body; small-caps with concentric-dot on eyebrows.
  • Blobs over illustration. Mint / sunshine / coral gradient blobs give ambient depth without literal leaves or stamps.
  • Wheel is more than a menu. The radial wheel Wednesday bonus: each spoke's icon + micro-label sells the Nintendo-pets game feel without any gacha-like currency.
