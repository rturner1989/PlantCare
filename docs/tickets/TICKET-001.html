TICKET-001

Install Dependencies + Design Tokens

Open High Priority

Summary

Install the frontend dependencies needed for the PlantCare design system (Framer Motion, Vaul, Fontsource fonts, driver.js) and configure the full Tailwind 4 theme with CSS custom properties matching the design spec. This is the foundation — every other ticket depends on these tokens and fonts being available.

Requirements

  1. Install npm packages inside Docker: motion, vaul, @fontsource/plus-jakarta-sans, @fontsource/fraunces, driver.js
  2. Rewrite client/src/index.css with Tailwind 4 @theme directive defining all palette colours, font families, corner radii, and shadow tokens
  3. Add :root CSS custom properties for direct var() access — gradient tokens (--gradient-mint, --gradient-forest), composite shadows (--shadow-fab, --shadow-dock)
  4. Import all required font weights via @fontsource: Plus Jakarta Sans (300, 400, 500, 600, 700, 800 + 500i, 700i) and Fraunces (400, 500, 600, 800 + all italic)
  5. Set body background to the mint gradient (var(--gradient-mint))
  6. Configure root element with Plus Jakarta Sans as default font family, optimised text rendering, and full-height flex column

Acceptance Criteria

docker compose exec client npm ls motion vaul driver.js shows all packages installed without errors
Plus Jakarta Sans renders at 800 weight in the browser (visually distinct from system sans-serif)
Fraunces renders at 800 italic weight in the browser (visually distinct serif italic)
Tailwind utility classes work with design tokens: bg-leaf, text-ink-soft, rounded-2xl, shadow-sm produce correct styles
CSS variables accessible: var(--leaf) resolves to #32C456, var(--gradient-mint) produces the vertical gradient
Browser at http://localhost:5173 shows the mint gradient background (not solid white)
./scripts/lint.sh passes with zero errors

Design Spec §3.3 — Typography

Plus Jakarta Sans — primary UI typeface. Weights: 300, 400, 500, 600, 700, 800. Lean bold — 700 and 800 are default for most text.

Fraunces — editorial display serif with beautiful italic. Desktop page titles and hero speech quotes only. NOT mobile body. Weights: 400, 500, 600, 800 with italic.

Design Spec §3.6 — Gradient Background

All authenticated screens use the mint gradient as the screen background:

background: linear-gradient(180deg, #F6FBF4 0%, #DFF2E2 50%, #C7ECCA 100%);

Content cards are white with mint borders, floating on the gradient.
